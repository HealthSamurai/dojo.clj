# Clojure Dojo

* Intro - what and why clojure?
* IDE & REPL: emacs / vim / vscode ....
* env: lein / deps.edn / docker
* koans as tests
* databases: jdbc, coersion, honeysql, hugsql etc
* web: httpkit, rest, routing, context
* ui: reagent, re-frame
* tests: back and ui
* deploy: jar, kubernetes


## Start

```sh
make up
make repl
# connect from ide


make build
make deploy

```
